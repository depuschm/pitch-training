# pitch-training
A [website](https://rayo3.github.io/pitch-training) designed for perfect pitch training.

### Technologies used
These technologies are used to develop this website:
- [MIDI.js](https://github.com/mudcube/MIDI.js) (used to play MIDI sounds)
